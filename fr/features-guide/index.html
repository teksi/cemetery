1. TEKSI cimetière (TCE)

TEKSI Cimetières est une sélection d’outils et un modèle de donnée qui vous permettent de :

  • Gérer et cartographier les données de votre cimetière

  • Indiquer les caractéristiques des objets du cimetières à travers des formulaires d’attributs

  • Produire des plans et extraire des statistiques de la base de données comme les contrats en cours, le nombre total de tombes, l’identification des prochaines interventions, etc.

  • Exporter les données dans divers formats

1.1. Fonctionnalités principales

Objets du cimetière

Création, information et gestion des éléments liés au cimetière

  • Tombes

  • Concessions

  • Columbariums

  • Ossuaires

  • Défunts

Visualisation et édition

Un projet QGIS préconfiguré incluant la symbologie, les formulaires et les relations entre les objets est fourni.

L’édition peut être effectuée directement en utilisant les outils QGIS natifs.

Fichiers

Les documents peuvent être liés aux éléments du cimetière.

Cartographie et production de statistiques

  • Map production with pre-configured layout

  • Serial plotting of structures protocole

  • Statistics production on all or part of the cemetery

Import de données

  • Field survey import tools (standard QGIS tools)

  • Data survey with a mobile device and semi-automatic insertion of the informations

Export de données

  • Data export in tab (Excel) and GIS format (all open data formats)

Historisation

  • Modifications history viewer and possibility to roll back

1.2. A venir

  • Modèles pour la gestion des documents

1.3. Limitations

Comme il n’existe à ce jour aucun standard officiel pour la gestion des cimetières, le module est basé sur les besoins utilisateurs et vise à être suffisamment flexible pour s’adapter si nécessaire.
